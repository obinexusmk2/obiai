# Security Policy

## Supported Versions
| Version | Supported          | Fuzzing gate |
|---------|--------------------|--------------|
| 1.0.x   | :white_check_mark: | MUST pass    |
| < 1.0   | :x:                | n/a          |

## Reporting a Vulnerability
1. **Encrypt** your report with the PGP key below.  
2. **Email** it to `security@obinexus.org` (or `security@obinexus.github.io` if no MX).  
3. **Include**:  
   - affected component (e.g., NPL parser, ATC coordinator)  
   - reproduction steps / minimal input file  
   - PoC code (if any)  
4. **Expect**:  
   - acknowledgement within 48 h  
   - regular status updates every 14 days  
   - public advisory **90 days** after first response (co-ordinated disclosure) unless mutually agreed otherwise.
